## 3. 響應式設計考慮
- 最大寬度: 800px
- 內邊距: 2rem
- 圖片區域高度: 300px
- 文字大小:
  - 標題: 2.5rem
  - 正文: 1.2rem
  - 按鈕: 1rem

## 4. 色彩方案
```
主要配色：
- 主色: #00f2ff (青色)
- 次要色: #ff00ff (洋紅)
- 背景色: #0a0a1a (深藍)
- 文字色: #ffffff (白色)
- 強調色: #ff3366 (紅色)

```
